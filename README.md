# dada-sorry 💗

A cute, animated "I'm Sorry / Please forgive me" page made for a best friend.

- Handwritten **Caveat** + soft **Quicksand** fonts, rose/peach theme
- Floating hearts background + confetti when forgiven 🎉
- A **"No" button that runs away** from your cursor (and your taps) so the only real answer is "Yes" 💗
- Fully responsive, accessible (respects `prefers-reduced-motion`)

## Personalize

Edit the names at the top of `src/App.jsx`, **or** pass them in the URL:

```
https://dada-sorry.vercel.app/?to=Riya&from=Dada
```

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm install -g vercel
vercel --prod          # name the project: dada-sorry
```

Built with React + Vite.
